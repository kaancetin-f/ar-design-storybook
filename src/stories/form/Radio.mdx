The Radio component is an interactive element used when users must select a single option from a predefined set. Radio buttons ensure that only one choice can be active at a time within the same group.

### Key Features

- **Single Selection:** Allows users to select only one option from a group.
- **Mutually Exclusive Choices:** Automatically deselects the previously selected option when a new one is chosen.
- **Clear Decision Making:** Ideal for forms and settings where exactly one option must be selected.

> **Note:** Use Radio Buttons when users must choose **only one** option from a list. If multiple options can be selected independently, use a **Checkbox** instead.
