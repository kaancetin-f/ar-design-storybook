The Button component is an interactive element used to trigger an action or submit a form.
Use buttons to enable users to take action with a single click.

```typescript

```

<h3>Group</h3>

Use the `ButtonGroup` component to group multiple buttons into a single container. This component automatically passes shared properties down to all inner buttons, ensuring UI consistency with less code.

```tsx
import { ButtonGroup } from "@harjs/react-ui";
```

<Canvas of={ButtonStories.Group} />

<h3>Action</h3>

The ButtonAction component is used to group multiple action buttons under a single dropdown menu (panel). When a trigger button, such as three dots (...), is pressed, it lists the buttons in the sub-panel. This component automatically passes shared properties to all buttons within it, providing a consistent user interface (UI) with less code.

```tsx
import { ButtonAction } from "@harjs/react-ui";
```
