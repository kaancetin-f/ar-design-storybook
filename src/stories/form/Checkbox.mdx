The Checkbox component is an interactive element used to select one or more options from a limited set. Use checkboxes to allow users to toggle independent choices or agree to single statements.

### Key Features

- **Multiple Selection:** Enables users to select multiple independent options from a list.
- **Binary State:** Ideal for single binary choices, such as "I agree to the terms and conditions."
- **Instant Feedback:** Can be used in filtering systems to update data dynamically upon selection.

> **Note:** Use Checkboxes when multiple options can be selected simultaneously. If the user must choose only _one_ option from a list, use a **Radio Button** instead.
