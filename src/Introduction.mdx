**A modern, fast, and completely independent UI components library.**

## Introduction

**HARJS** is a modern UI library optimized for high-performance web applications. It is designed to maximize the developer experience (DX) while delivering a smooth and responsive interface for end users.

> ⚡ **Key Advantage: Zero Dependencies**
>
> HARJS is built entirely from scratch using **pure/native** web technologies, without relying on any third-party packages. This keeps your project's bundle size lean, minimizes security risks (no more npm audit nightmares), and delivers optimal runtime performance.

---

## Key Features

- **Zero Dependencies:** No third-party packages. Lightweight, secure, and reliable.
- **Accessible:** Built with WAI-ARIA standards in mind.
- **Performance-Focused:** Minimal bundle size with maximum rendering performance.
- **Highly Customizable:** Easily adapt the library to your brand using CSS Custom Properties.
